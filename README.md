# @chrisbarrett/skillful

[![JSR](https://jsr.io/badges/@chrisbarrett/skillful)](https://jsr.io/@chrisbarrett/skillful)

Deno library for incrementally discovering and activating
[Claude Code](https://code.claude.com) skills based on files referenced and
keywords in prompts.

For the full documentation, check the
[latest release on JSR](https://jsr.io/@chrisbarrett/skillful@latest).
